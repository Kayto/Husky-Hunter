# HBA — Short Programs

Shorter stand-alone Hunter BASIC programs. Each is self-contained and
intended to run directly on the Husky Hunter via HCOM transfer.

---

| File | Description |
| ---- | ----------- |
| `CHARSET.HBA` | Prints the printable character set (ASCII 32–127) one row per 32 chars |
| `COLLATZ.HBA` | Collatz conjecture: enter a starting number, prints sequence and step count |
| `HELLO.HBA` | Hello World — full-screen graphic title using `SCREEN 1`, large font, press any key to exit |
| `SYSINFO.HBA` | Hunter diagnostic: memory, power-off inhibit, display/keyboard state, serial port settings, baud/serial-mode actions. Paged to fit 8-line LCD |
