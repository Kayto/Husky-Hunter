# BASIC — Short Programs

Shorter stand-alone Hunter BASIC programs.

---

| File | Description |
| ---- | ----------- |
| `CHARSET.BAS` | Prints the printable character set (ASCII 32–127) one row per 32 chars |
| `COLLATZ.BAS` | Collatz conjecture: enter a starting number, prints sequence and step count |
| `HELLO.BAS` | Hello World — full-screen graphic title using `SCREEN 1`, large font, press any key to exit |
| `SYSINFO.BAS` | Hunter diagnostic: memory, power-off inhibit, display/keyboard state, serial port settings, baud/serial-mode actions. Paged to fit 8-line LCD |
